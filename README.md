# MungNyang Market

반려동물 쇼핑몰을 주제로 제작한 반응형 퍼블리싱 포트폴리오입니다.

베스트 상품 슬라이드, 상품 카테고리 필터, 드롭다운 메뉴, 모바일 햄버거 메뉴를 구현했고, 상업형 UI 구조를 중심으로 쇼핑몰 메인페이지를 구성했습니다.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- 반응형 레이아웃
- 드롭다운 네비게이션 메뉴
- 모바일 햄버거 메뉴
- 베스트 상품 슬라이드
- 상품 카테고리 필터
- 리뷰 카드 섹션
- 문의 / 브랜드 소개 섹션

## Project Description
실제 반려동물 쇼핑몰처럼 보이도록 메인 비주얼, 베스트 상품, 상품 리스트, 프로모션, 리뷰, 브랜드 소개, 문의 섹션으로 구성했습니다.  
사용자 입장에서 한눈에 상품을 탐색할 수 있도록 카드형 UI와 카테고리 분류를 적용했습니다.
